My Name is Daniel, welcome to my portfolio

Multimedia Engineer and self taught web developer, hoping to get certified to land a job.

These are projects I worked on my own in order to get more confident with my skills and learn more in the process. Most of them are part of the freeCodeCamp program.

The Simpsons Quotes

I made a quote generator using quotes from The Simpsons (mostly Homer). This project uses a modification of aitorp6's API, and is part of the freeCodeCamp program. I used Bootstrap and Pug

Weather app

As part of the freeCodeCamp program, I built an app that fetches the weather of the user's location. Browsers require a secure connection for geolocalization, so I had to use CloudFlare's services to get the full site to SSL.

Wikipedia Viewer

I built a Wikipedia Viewer using MediaWiki's API, Bootstrap and Pug. Users can search an article and get the main results or get a random article.

Jekyll Blog

I'm currently populating with content a Jekyll blog that I tested locally and then deployed on surge.sh. I'm glad I found out about this tool, it's a efficient and fast way to get a site published online. And it served as excuse to install and start using npm.

Tribute Page

This was the first project of the freeCodeCamp program. It's a tribute page using Bootstrap. I made it about Tame Impala.

Twitch API Project

I made a small app using the Twitch API as part of the freeCodeCamp program. It displays the status and image from a couple of channels. I used Materialize and Pug.
